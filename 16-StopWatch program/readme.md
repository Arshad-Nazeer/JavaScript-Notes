# JavaScript Stopwatch

## 📚 Features

* Start, stop, and reset the stopwatch
* Displays time in **hours : minutes : seconds : milliseconds**
* Real-time updates using JavaScript timers
* Responsive and clean user interface
* Interactive buttons with hover effects

## 🛠️ Concepts Practiced

* DOM Manipulation
* Event Handling
* `setInterval()` and `clearInterval()`
* Time calculations
* Dynamic content updates
* HTML, CSS, and JavaScript integration

## 🎯 Outcome

Built a functional stopwatch that tracks elapsed time accurately while strengthening core JavaScript concepts such as timers, DOM manipulation, and user interaction.
