### 🎮 Number Guessing Game

A simple browser-based game where the user guesses a randomly generated number within a given range. The program validates user input, provides hints ("Too High" or "Too Low"), tracks the number of attempts, and allows the user to exit by pressing **Cancel**.

**Concepts Used:**

* `Math.random()` & `Math.floor()`
* `while` loop
* `break`
* Conditional statements (`if...else`)
* User input with `window.prompt()`
* Input validation (`isNaN()`)
* Type conversion (`Number()`)
* Attempt counter
