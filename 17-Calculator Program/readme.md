# 🧮 JavaScript Calculator

A simple calculator built using **HTML**, **CSS**, and **JavaScript**. It performs basic arithmetic operations with a clean, responsive interface.

## ✨ Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Decimal number support
- Clear (`C`) button
- Error handling for invalid expressions

## 🛠️ Technologies Used
- HTML5
- CSS3 (Grid Layout & Flexbox)
- JavaScript (DOM Manipulation, Functions, `eval()`)

## 📂 Project Structure
```
calculator/
├── index.html
├── style.css
├── script.js
└── README.md
```

## 📸 Preview
A minimal calculator with a dark theme, circular buttons, and orange operator keys.

## 📚 Concepts Practiced
- DOM Manipulation
- Event Handling (`onclick`)
- CSS Grid
- Functions
- `try...catch`
- `eval()` for expression evaluation

## 🚀 How to Run
1. Clone or download the project.
2. Open `index.html` in your browser.
3. Start calculating!
