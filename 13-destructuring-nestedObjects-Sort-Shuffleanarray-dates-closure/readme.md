# JavaScript Learning – Day 7

## 📚 Topics Covered

* Array & Object Destructuring
* Nested Objects and Classes
* Arrays of Objects
* Array Methods (`forEach`, `map`, `filter`, `reduce`)
* Sorting Arrays with `sort()`
* Custom Comparison Functions
* Lexicographic Sorting using `localeCompare()`
* Array Shuffling (Random Sort vs. Fisher–Yates Algorithm)
* Date Objects (`Date`, Getters & Setters)
* Closures and Private State

## 💡 Mini Examples

* Swapping variables using destructuring
* Extracting object properties and function parameter destructuring
* Modeling nested objects with classes
* Manipulating arrays of objects
* Sorting numbers, strings, and objects
* Implementing an unbiased Fisher–Yates shuffle
* Creating and modifying JavaScript `Date` objects
* Building counters and score trackers using closures

## 🎯 Key Takeaways

* Destructuring makes code cleaner and more readable.
* `sort()` requires a custom comparison function for numbers and objects.
* `cards.sort(() => Math.random() - 0.5)` is **not** a proper shuffle—use the Fisher–Yates algorithm instead.
* JavaScript `Date` objects can be created, formatted, and modified using getter and setter methods.
* Closures preserve variables from an outer function, enabling private state and encapsulation.
