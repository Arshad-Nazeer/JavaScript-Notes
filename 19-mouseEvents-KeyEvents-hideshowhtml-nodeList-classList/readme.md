# JavaScript DOM Events & Manipulation

This project demonstrates how to use JavaScript to create interactive web pages by handling user events and manipulating the DOM.

## Topics Covered

- Mouse Events
  - `click`
  - `mouseover`
  - `mouseout`
  - `addEventListener()`
  - Event object (`event`, `event.target`)

- Keyboard Events
  - `keydown`
  - `keyup`
  - Reading `event.key`
  - Moving elements with arrow keys

- Show / Hide Elements
  - `display: none`
  - `visibility: hidden`

- NodeLists
  - `querySelectorAll()`
  - Iterating with `forEach()`
  - Static nature of NodeLists
  - Adding and removing elements dynamically

- Creating & Removing Elements
  - `createElement()`
  - `append()`
  - `remove()`

- `classList`
  - `add()`
  - `remove()`
  - `toggle()`
  - `replace()`
  - `contains()`

## Mini Demonstrations

- Interactive color-changing box
- Button-controlled event handling
- Keyboard key detector
- Arrow key movement
- Image hide/show toggle
- Dynamic button creation and deletion
- Reusable CSS classes with `classList`

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
