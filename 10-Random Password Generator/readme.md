# Password Generator

A simple JavaScript password generator with a retro pixel-themed interface.

## Features

* Generate passwords of any length
* Include:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Symbols
* Input validation for invalid length or no character set selected
* Instant password generation

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Enter the desired password length.
2. Select the character types to include.
3. Click **Generate**.
4. Copy the generated password displayed on the screen.
