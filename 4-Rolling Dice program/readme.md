### 🎲 Dice Roller

A simple dice rolling application that simulates rolling three six-sided dice. Each click of the **Roll** button generates three random numbers between 1 and 6 and displays the results on the webpage.

**Concepts Used:**

* `Math.random()` & `Math.floor()`
* Random Number Generation
* DOM Manipulation (`getElementById()`)
* Event Handling (`onclick`)
* Updating HTML with `.textContent`
