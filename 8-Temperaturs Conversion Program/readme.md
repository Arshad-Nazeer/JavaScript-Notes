# 🌡️ Temperature Converter

A simple temperature converter built using **HTML, CSS, and JavaScript**.

## Features

* Convert **Celsius → Fahrenheit**
* Convert **Fahrenheit → Celsius**
* User-friendly interface
* Rounded results using `toFixed(1)`
* Responsive card layout with basic styling and hover effects

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## How It Works

1. Enter a temperature in the input box.
2. Select the desired conversion:

   * Celsius → Fahrenheit
   * Fahrenheit → Celsius
3. Click the **Submit** button.
4. The converted temperature is displayed below the button.

## Conversion Formulas

* **Celsius to Fahrenheit**

  `°F = (°C × 9/5) + 32`

* **Fahrenheit to Celsius**

  `°C = (°F − 32) × 5/9`
