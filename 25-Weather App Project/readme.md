# Weather App 🌤️

A simple weather application built with **HTML, CSS, and JavaScript** that fetches real-time weather data using the **OpenWeatherMap API**.

## Features

* Search weather by city name.
* Displays:

  * 🌡️ Temperature
  * 💧 Humidity
  * 📝 Weather description
  * 🌤️ Weather emoji based on weather conditions
* Handles invalid city names and empty input.
* Clean, responsive interface.

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeatherMap API

## How to Run

1. Clone or download the repository.
2. Replace the API key in `script.js` with your own OpenWeatherMap API key.
3. Open `index.html` in your browser.
