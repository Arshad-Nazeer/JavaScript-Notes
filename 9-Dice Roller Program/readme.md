# JavaScript Dice Roller

## 📚 Features

* Roll one or multiple dice at once
* Generates random values between **1 and 6**
* Displays the rolled numbers instantly
* Shows corresponding dice images for each roll
* Simple and responsive user interface

## 🛠️ Concepts Practiced

* DOM Manipulation
* Event Handling
* `Math.random()` and `Math.floor()`
* Arrays (`push()`, `join()`)
* Loops
* Dynamic HTML generation with `innerHTML`

## 🎯 Outcome

Built an interactive dice roller that generates random dice values and dynamically updates the webpage with both the numerical results and matching dice images.
