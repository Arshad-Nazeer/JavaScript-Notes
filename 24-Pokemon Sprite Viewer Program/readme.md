# Pokémon Sprite Program ⚡

A simple Pokémon search application built with **HTML, CSS, and JavaScript** that fetches Pokémon data using the **PokéAPI**.

## Features

* Search Pokémon by name.
* Displays the Pokémon's official front sprite.
* Fetches real-time data using the Fetch API.
* Handles invalid Pokémon names.
* Simple and responsive interface.

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* PokéAPI

## How to Run

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Enter a Pokémon name and click **Fetch Pokemon**.
