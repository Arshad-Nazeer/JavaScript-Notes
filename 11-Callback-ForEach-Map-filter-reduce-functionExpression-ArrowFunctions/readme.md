# Day 5 - JavaScript Array Methods & Functions

## 📚 Topics Covered

### 🔹 Callbacks
- Functions passed as arguments to other functions.
- Commonly used for asynchronous operations.
- Examples:
  - Reading files
  - Network requests
  - Database operations

### 🔹 `forEach()`
- Iterates through each element of an array.
- Modifies the original array.
- Common uses:
  - Display elements
  - Transform array values

### 🔹 `map()`
- Creates a **new array** by applying a callback to every element.
- Does **not** modify the original array.
- Useful for data transformation.

### 🔹 `filter()`
- Creates a new array containing only elements that satisfy a condition.
- Examples:
  - Even/Odd numbers
  - Adults/Children

### 🔹 `reduce()`
- Reduces an array to a single value.
- Common uses:
  - Sum of elements
  - Finding maximum/minimum values

### 🔹 Function Declaration
- Standard way to define reusable functions.

### 🔹 Function Expressions
- Functions stored inside variables.
- Can be passed directly as callbacks.
- Includes anonymous functions.

### 🔹 Arrow Functions (`=>`)
- Shorter syntax for function expressions.
- Best for simple, one-time callbacks.
- Often used with:
  - `map()`
  - `filter()`
  - `reduce()`
  - `setTimeout()`

## 🛠️ Concepts Practiced
- Passing callbacks
- Array iteration
- Array transformation
- Array filtering
- Array reduction
- Function declarations vs expressions
- Anonymous functions
- Arrow functions
- Callback-based programming
