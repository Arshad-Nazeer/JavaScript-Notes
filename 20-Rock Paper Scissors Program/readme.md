# Rock Paper Scissors 🎮

A simple Rock-Paper-Scissors game built with **HTML, CSS, and JavaScript**.

## Features

* Play against a computer with random choices.
* Displays both player and computer selections.
* Tracks player and computer scores.
* Highlights the game result:

  * 🟢 Green for a win
  * 🔴 Red for a loss
* Clean, responsive Flexbox-based layout.

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)

## How to Run

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Click a button to play.
