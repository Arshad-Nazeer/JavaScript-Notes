# JavaScript DOM Manipulation

## 📚 Topics Covered

* Document Object Model (DOM)
* The `document` object
* Accessing and modifying HTML elements
* Element Selectors:

  * `getElementById()`
  * `getElementsByClassName()`
  * `getElementsByTagName()`
  * `querySelector()`
  * `querySelectorAll()`
* HTMLCollection vs NodeList
* DOM Navigation:

  * `firstElementChild`
  * `lastElementChild`
  * `nextElementSibling`
  * `previousElementSibling`
  * `parentElement`
  * `children`
* Creating, inserting, moving, and removing DOM elements
* Dynamic styling with JavaScript

## 🛠️ Practice

* Changed webpage title and theme dynamically.
* Selected and styled elements using various DOM selectors.
* Traversed the DOM using parent, child, and sibling relationships.
* Created, inserted, repositioned, and removed HTML elements.
* Worked with HTMLCollections and NodeLists.

## 🎯 Outcome

Built a strong foundation in DOM manipulation, enabling JavaScript to dynamically modify a webpage's content, structure, and appearance.
