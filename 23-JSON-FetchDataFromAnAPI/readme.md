# JavaScript JSON & Fetch API

This project demonstrates how JavaScript works with **JSON**, **HTTP requests**, and the **Fetch API** to retrieve and process data asynchronously.

## Topics Covered

- JSON objects and arrays
- `JSON.stringify()` and `JSON.parse()`
- Fetch API
- HTTP requests and responses
- Response object and `response.ok`
- Fetching local JSON files
- Fetching data from the PokéAPI
- Promise chaining with `.then()`
- Error handling with `.catch()`
- Async/Await with `try...catch`

## Concepts Learned

- Converting JavaScript objects ↔ JSON strings
- Parsing JSON responses
- Making asynchronous HTTP requests
- Working with the Response object
- Handling API and network errors
- Consuming REST APIs using Fetch

---

**Learning Outcome:** Learned how to exchange JSON data, fetch resources from servers and APIs, and handle asynchronous operations using Promises and Async/Await.
