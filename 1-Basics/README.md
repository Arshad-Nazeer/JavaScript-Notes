# JavaScript Basics

This folder contains my notes and practice code for the fundamental concepts of JavaScript. It serves as the foundation for the rest of my JavaScript learning journey.

## Topics Covered

* Console Output

  * `console.log()`
  * `window.alert()`

* Comments

  * Single-line comments (`//`)
  * Multi-line comments (`/* */`)

* Strings

  * Single quotes (`' '`)
  * Double quotes (`" "`)
  * Template literals (`` ` ` ``)

* Variables

  * `var`
  * `let`
  * `const`

* Data Types

  * Primitive data types
  * Non-primitive (reference) data types
  * `typeof` operator

* Numbers

  * Integer and floating-point values
  * Displaying numeric values

* Booleans

  * `true` and `false`

* DOM Manipulation

  * `document.getElementById()`
  * Updating text using `.textContent`

* Arithmetic Operators

  * `+`, `-`, `*`, `/`, `%`, `**`
  * `++`, `--`
  * Compound assignment operators

* Operator Precedence

* User Input

  * `window.prompt()`
  * HTML input fields
  * Button click events (`onclick`)

* Type Conversion

  * `Number()`
  * `String()`
  * `Boolean()`

* Constants

  * Using `const`
  * Circumference calculator project

## Mini Projects

* Username input using an HTML form
* Circle circumference calculator

## Purpose

The purpose of this folder is to document my understanding of JavaScript fundamentals through notes, examples, and small practice programs. As I continue learning, more advanced topics will be organized into separate folders within this repository.
