# Day 6 – JavaScript OOP

Today I learned the fundamentals of Object-Oriented Programming (OOP) in JavaScript.

## Topics Covered
- Objects and object methods
- The `this` keyword
- Constructor functions
- ES6 Classes
- Static properties and methods
- Inheritance using `extends`
- The `super` keyword
- Getters and Setters
- Basic data validation using setters

## Key Takeaways
- Classes provide a cleaner way to create objects than constructor functions.
- `this` refers to the current object executing a method.
- Static members belong to the class, not its instances.
- Inheritance promotes code reusability.
- `super()` calls the parent constructor, while `super.method()` calls a parent method.
- Getters and setters help encapsulate data and validate values before storing them.
